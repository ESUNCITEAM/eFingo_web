常見問題

    e.Fingo會員 熊朋友邀請賽

Q1 / e.Fingo是什麼？

e.Fingo是玉山銀行全新推出的數位會員！串聯既往深受顧客好評的玉山行動銀行App、玉山Wallet App、e指系列服務，提供美好數位金融體驗、為顧客生活帶來更多精彩。加入e.Fingo數位會員可享有一系列的專屬會員活動及相關權益，將陸續推出，敬請期待！

Q2 / 誰可以加入e.Fingo會員？

    加入e.Fingo 會員需具備以下 2 個條件，並透過玉山行動銀行App加入：

  • 持有「臺幣帳戶（含數位帳戶）」
  • 持有「玉山 U Bear 信用卡／玉山 Only 卡／玉山 Pi 拍錢包信用卡／玉山 Pi 拍兔 Debit卡」任一張指定卡片

    • （提醒您：1. 請將玉山行動銀行App更新至最新版本。 2. 如為新開戶或新核卡，請於次一工作日加入會員！3.現階段僅提供本國自然人申請。）

Q3 / e.Fingo會員專區在哪裡？

顧客可以在玉山行動銀行App側邊欄，或是首頁介紹卡片找到e.Fingo會員專區喔！

Q4 / 如果不小心刪除玉山行動銀行App，我的會員資格會被取消嗎？

不會喲！請重新下載玉山行動銀行App，透過會員專區參與會員活動、體驗便利的數位服務。

Q5 / 什麼情形會喪失會員資格？

如果顧客未持有玉山臺幣帳戶及任一張玉山信用卡，則無法再使用e.Fingo數位會員服務。

Q1 / 如果我這個月中獎了，我下個月抽獎機會會歸零嗎？

不會喔！《熊朋友邀請賽》活動抽獎機會將持續累積至當季活動結束，如果本月中獎，抽獎機會仍會持續累積至次月！

Q2 / 邀請人數有限制嗎？

沒有喔！邀請越多好友、可獲得越多抽獎機會！

Q3 / 如果好友已經加入會員，但他沒有填過好友邀請碼，可以再補填我的好友邀請碼嗎？

不行喔！已經加入的會員就沒有辦法再補填寫好友邀請碼囉~請邀請其他新會員加入!

Q4 / 顧客加入會員之後，可以更改已填寫的好友邀請碼嗎？

不行喔！好友邀請碼填寫送出後便無法進行修改，也沒有辦法進行補填喔~
